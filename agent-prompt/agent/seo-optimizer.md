After completing development, audit the entire project and automatically CREATE, UPDATE, or FIX all important production files related to SEO, indexing, crawling, discoverability, social sharing, performance, and search engine visibility.

Requirements:

1. Crawling & Indexing
- Generate/update robots.txt
- Generate/update sitemap.xml
- Generate sitemap index if multiple sitemaps exist
- Add image sitemap if images exist
- Add video sitemap if videos exist
- Add news sitemap if applicable
- Verify robots rules do not block important pages
- Ensure canonical URLs exist
- Add noindex where required
- Validate crawl paths

2. Metadata & SEO
- Ensure every page has:
  - title tag
  - meta description
  - canonical URL
  - Open Graph tags
  - Twitter/X card metadata
  - robots meta tag
  - language declaration
  - viewport tag
  - charset
- Generate dynamic metadata where needed
- Detect duplicate metadata
- Fix missing metadata

3. Structured Data
Generate JSON-LD structured data where applicable:
- Organization
- Website
- BreadcrumbList
- Product
- Article
- FAQ
- LocalBusiness
- SoftwareApplication
- BlogPosting
- Person
- Review
- Service
- Event
- VideoObject

Validate schema implementation.

4. Discovery Files
Create/update:
- robots.txt
- sitemap.xml
- sitemap-index.xml
- manifest.json
- browserconfig.xml
- humans.txt
- security.txt
- ads.txt (if ads exist)
- rss.xml / feed.xml if content/blog exists

5. PWA & App Discovery
Verify:
- web manifest
- icons
- favicon variants
- apple-touch-icon
- mask-icon
- theme-color
- app metadata

6. Social Sharing Optimization
Ensure:
- Open Graph image generation
- Twitter card image
- social preview validation
- fallback sharing image

7. Performance SEO
Audit:
- image optimization
- lazy loading
- font loading
- script loading
- preload critical assets
- cache headers
- compression support
- Core Web Vitals impact
- CLS prevention
- LCP optimization
- JS bundle size

8. Internal SEO Architecture
Verify:
- breadcrumbs
- internal linking
- pagination metadata
- hreflang if multilingual
- canonical consistency
- redirect chains
- orphan pages
- broken links

9. Technical SEO
Check:
- HTTPS enforcement
- redirect www/non-www consistency
- trailing slash consistency

- 404 page
- 410 handling
- XML validity
- robots validity

- mobile friendliness

10. AI / LLM Discovery Optimization
Create/update:
- llms.txt
- llms-full.txt if supported
- AI-readable content summaries
- semantic metadata
- machine-readable site descriptions

11. Search Console & Analytics Readiness
Verify placeholders/config:
- Google Search Console verification
- Bing Webmaster verification
- analytics integration
- event tracking
- conversion tracking

12. Final Validation
Produce report:
✓ files created
✓ files updated
✓ SEO issues fixed
✓ crawl issues fixed
✓ schema added
✓ warnings remaining

Prioritize production readiness, search visibility, crawler compatibility, AI discoverability, and long-term maintainability.
